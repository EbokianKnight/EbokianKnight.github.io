Adflixs

I produced a single-page website called Adflixs in a two-week sprint. Adflixs is a web application built to showcase video advertisements and track user interest. I handled user database queries through JSON using Ajax requests and throttled large requests by implementing infinite scrolling. It was inspired by the design of Netflix and built using Ruby on Rails, Javascript, jQuery, React, and Flux. I learned Heroku to host the database and supplemented large file storage using Amazon Web Services.

Ebokian Server

I love working with Ruby, and found that too much of Rails felt like magic. Using TTD, I took it apart, learned the features and built my own RESTful MVC framework called EbokianServer. Implementing some ruby meta-programming tricks I made my own dynamic ORM tools to turn SQL requests into modeled ruby objects. Building on top of the rack middleware, I created router switch board to decode http params between the user and the server through the controllers. Finally I connected the controllers to the models and implemented an SQL database so the entire project could be put to work.
